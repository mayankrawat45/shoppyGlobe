# ShoppyGlobe E-commerce Application

A responsive e-commerce application built using React, Redux Toolkit, React Router, Tailwind CSS, and Vite.

## Features

- Product listing from API
- Dynamic product detail page
- Add to cart functionality
- Quantity management
- Redux state management
- Product search
- Checkout flow
- Lazy loading
- Responsive design
- Error handling
- 404 page

## Tech Stack

- React
- Redux Toolkit
- React Router DOM
- Tailwind CSS
- Vite

## API Used

https://dummyjson.com/products

## Run Locally

```bash
npm install
npm run dev
```
